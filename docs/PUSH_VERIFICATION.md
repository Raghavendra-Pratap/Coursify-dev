## After clone: minimal steps to run

```bash
git clone https://github.com/Raghavendra-Pratap/Coursify-dev.git
cd Coursify-dev
git checkout develop
cp env.template .env.local   # then fill in Supabase + optional Resend
npm install
npm run build
npm run start
```

If you use **coursify-app** as a separate app: `cd coursify-app && npm install` there too.
